# Frequently Asked Questions (FAQ)

## General Questions

### What is ClassX?
ClassX is an AI-powered learning platform designed to help beginners master programming skills through interactive courses, hands-on projects, and personalized AI tutoring.

### Who is ClassX for?
ClassX is designed for:
- Programming beginners
- Students learning to code
- Career changers entering tech
- Anyone wanting to improve their programming skills

### Is ClassX free?
Yes! ClassX is completely free and open-source. We believe education should be accessible to everyone.

### What programming languages does ClassX support?
Currently, ClassX supports:
- JavaScript
- TypeScript
- React
- Node.js
- Python
- HTML/CSS
- And more coming soon!

## Technical Questions

### What are the system requirements?
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- No additional software required

### Can I use ClassX offline?
Currently, ClassX requires an internet connection as it uses AI-powered features and cloud-based resources.

### Is my data secure?
Yes! We take security seriously. All data is encrypted and we follow industry best practices for data protection. See our [Security Policy](../SECURITY.md) for more details.

### How does the AI tutor work?
Our AI tutor uses advanced machine learning to:
- Analyze your learning progress
- Provide personalized recommendations
- Answer questions in real-time
- Adapt to your learning style

## Account & Usage

### Do I need to create an account?
While you can browse some content without an account, creating an account allows you to:
- Track your progress
- Save your work
- Access personalized features
- Join the community

### How do I reset my password?
1. Go to the login page
2. Click "Forgot Password"
3. Enter your email address
4. Check your email for reset instructions

### Can I delete my account?
Yes, you can delete your account at any time. Go to Settings > Account > Delete Account.

### Is there a mobile app?
Currently, ClassX is web-based and works great on mobile browsers. A native mobile app is planned for the future.

## Learning & Courses

### How do I start learning?
1. Sign up for a free account
2. Choose your learning path
3. Start with the basics
4. Complete hands-on projects
5. Track your progress

### Are there prerequisites?
No! ClassX is designed for complete beginners. We start from the very basics.

### How long does it take to complete a course?
Course duration varies:
- Basic courses: 2-4 weeks
- Intermediate courses: 4-8 weeks
- Advanced courses: 8-12 weeks

You can learn at your own pace!

### Can I skip ahead?
Yes! You can skip sections you already know, but we recommend following the structured path for the best learning experience.

### Are there certificates?
Yes! You'll receive certificates upon completing courses and projects.
